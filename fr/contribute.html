Participer

Ce site a été conçu avec les données de certains conseils locaux et de régions du Cameroun, mais il reste encore du travail à faire pour compléter le tableau des dépenses du Cameroun et vous pouvez aider. Que vous soyez un conseiller municipal ou un citoyen, impliquez-vous et transmettez les données de votre région!

Impliquez-vous

N'importe qui peut fournir des données de son conseil local, les visualiser et les relier à la carte. L'objectif est d'obtenir une image aussi complète que possible des finances du Cameroun et de continuer à construire ce site de référence, année après année. Voici comment faire:

Transformez vos données au format OpenSpending

Suivez les conseils du manuel pour savoir quel objectif viser. Vous pouvez jeter un oeil aux templates pour vous faire une idée de ce à quoi les données doivent ressembler.

Certaines modifications peuvent être faites dans un programme de tableur comme Excel. Cependant, certains outils plus puissants comme Google Refine peuvent vous aider à traiter de gros volumes de données. Lisez le tutorial de Google Refine pour plus d'informations.

Mettez vos données en ligne

Ensuite, vous devrez mettre en ligne votre fichier CSV. Ils y a plusieurs manière de faire cela, comme Dropbox ou en exportant vos données directement depuis Google Docs (Fichier > Publier sur le Web > Publier CSV). Nous avons utilisé le DataHub pour publier nos données. Si vous ne l'avez jamais fait, voici une courte explication de comment mettre en ligne un jeu de données.

La vidéo ci-dessous explique comment réaliser l'opération:

Mettre en ligne des données sur le DataHub présenté par l'Open Knowledge Foundation sur Vimeo.

Importez vos données vers OpenSpending

Ensuite, ajoutez des informations sur votre jeu de données pour le rendre facilement trouvable dans OpenSpending. Encore une fois, le tutoriel vidéo ci-dessous vous guide pas-à-pas:

Mise en ligne sur OpenSpending: Etape 1 présenté par l'Open Knowledge Foundation sur Vimeo.

De plus amples informations sont disponibles sur le site d'OpenSpending.

Créez un modèle pour vos données

Ensuite, dites à OpenSpending comment comprendre vos données en créant un modèle.

Mise en ligne sur OpenSpending: Etape 2 - Créer un modèle présenté par l'Open Knowledge Foundation sur Vimeo.

Si c'est tout ce passe bien, vous serez en mesure de passer à la création de visualisations (vidéo à venir bientôt - on vous tient au courant). Si vous obtenez des erreurs, regardez la vidéo ci-dessous pour savoir quoi faire.

Réparer les erreurs

Tout ne fonctionne pas forcément parfaitement la première fois que vous essayez de mettre en ligne vos données - voici ce qu'il faut faire si vous avez fait une erreur, que ce soit dans votre modèle ou dans vos données.

Comment réparer les erreurs dans OpenSpending présenté par l'Open Knowledge Foundation sur Vimeo.

Visualisations

Maintenant que vous avez terminé, des visualisations pourront être automatiquement générées à partir des données que vous avez mis en ligne sur OpenSpending. Tout ce que vous avez à faire est:

  1. Lier les données à Cameroon.OpenSpending.org. Vous trouverez ici un tutorial
  2. Si vous ajoutez des données à une nouvelle région du Cameroun (par opposition à de nouvelles données relatives à un nouvel exercice d'une région du Cameroun qui dispose déjà d'une carte) - vous aurez besoin de créer une nouvelle sous-carte. Vous trouverez les instructions sur comment procéder en cliquant ici.
